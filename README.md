# 🎬 Movie Analytics Dashboard (Power BI)

This project is a professional Power BI dashboard built using a movie dataset.

## 📊 Dashboard Features:

* Total Movies, Revenue, Votes, Avg Rating
* Genre-wise analysis
* Year-wise movie trends
* Interactive slicers (filters)

## 🛠 Tools Used:

* Power BI Desktop
* Dataset: movies_updated.csv

## 📁 Files Included:

* MovieDashboard.pbix
* movies_updated.csv

## 🚀 Key Insights:

* Most popular movie genres
* Revenue distribution by genre
* Movie trends over time
